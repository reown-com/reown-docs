To open Reown you can use our default [web components](/appkit/vue/core/components) or build your own logic using [Reown composables](/appkit/vue/core/composables).
In this example we are going to use the `<appkit-button>` component.

Web components are global html elements that don't require importing.

```vue
<template>
  <appkit-button />
</template>
```
