To open AppKit you can use our default [web components](/appkit/vue/core/components) or build your own logic using [AppKit composables](/appkit/vue/core/composables).
In this example we are going to use the `<appkit-button>` component.

Web components are global html elements that don't require importing.

```vue
<template>
  <client-only>
    <appkit-button />
  </client-only>
</template>
```
