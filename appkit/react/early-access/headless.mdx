## Hooks

To start building custom connection user interfaces, you can use the following hooks.

## useAppKitWallets

The essential hook for accessing all type wallets and methods to make connections.

### Use Cases

- Get available injected wallets
- Get, search, and paginate WalletConnect wallets
- Connect to injected wallets
- Get WalletConnect QR URI

### Returns

| Return Value      | Type                                                        | Description                                                                                                                                      |
| ----------------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| data              | WalletItem[]                                                | List of all wallets (injected wallets and WalletConnect wallets combined).                                                                       |
| isFetchingWallets | boolean                                                     | Boolean indicating if WalletConnect wallets are being fetched.                                                                                   |
| isFetchingWcUri   | boolean                                                     | Boolean indicating if a WalletConnect URI is being fetched.                                                                                      |
| wcUri             | string \| undefined                                         | The current WalletConnect URI for QR code display (set when connecting to a WalletConnect wallet).                                               |
| page              | number                                                      | The current page number of WalletConnect wallet results.                                                                                         |
| count             | number                                                      | The total number of available wallets (for pagination/counting).                                                                                 |
| fetchWallets      | (options) => Promise                                        | Function to fetch WalletConnect wallets from the explorer API. Enables pagination/initial load.                                                  |
| connect           | (wallet: WalletItem, namespace?: ChainNamespace) => Promise | Function to connect to a wallet. For WalletConnect wallets: starts connection & updates wcUri. For injected wallets: triggers direct connection. |

## Example Use Cases

### List and Connect to Injected Wallets

You could render the available injected wallet detected in your browser.

If you configure your AppKit as multi-chain, we suggest rendering a namespace selection component where users should select which namespace to connect. This could be achieved with the `connectors` array of the wallet item.

<CodeGroup>

```tsx InjectedWallets.tsx
const { data } = useAppKitWallets()

const injectedWallets = data.filter(wallet => wallet.isInjected)

injectedWallets.map(wallet => {
  return <WalletItem wallet={wallet} />
})
```

```tsx WalletItem.tsx
export function WalletItem({ wallet }: { wallet: WalletItem }) {
  const { connect } = useAppKitWallets()

  async function handleConnect(wallet: WalletItem, namespace?: ChainNamespace) {
    await connect(wallet, namespace)
      .then(() => {
        toast({ title: 'Connected', status: 'success' })
      })
      .catch(() => {
        toast({ title: 'Connection declined', status: 'error' })
        setConnectingWallet(undefined)
      })
  }

  return (
    <div>
      <span>{wallet.name}</span>
      {wallet.connectors.map(item => {
        return <button onClick={() => handleConnect(wallet, item.chain)}>{item.chain}</button>
      })}
    </div>
  )
}
```

</CodeGroup>

### Render and Connect to WalletConnect Wallets

You can render all WalletConnect wallets and generate QR code to connect to them.

<CodeGroup>

```tsx WalletConnectWallets.tsx
const { data, isFetchingWcUri, isFetchingWallets, page, count, fetchWallets } = useAppKitWallets()
const [inputValue, setInputValue] = useState('')
const searchQuery = useDebounceValue(inputValue, 500)

// Filter wallets as non injected ones.
const wcWallets = data.filter(w => !w.isInjected)

useEffect(() => {
  fetchWallets?.()
}, [])

// Query WalletConnect wallets
useEffect(() => {
  if (searchQuery.length > 0) {
    fetchWallets?.({ query: searchQuery })
  } else {
    fetchWallets?.()
  }
}, [searchQuery])

return wcWallets.map(wallet => {
  return (
    <WalletItem
      name={wallet.name}
      onClick={() => {
        handleWallet(wallet)
      }}
      imageSrc={wallet.imageSrc}
      badge={wallet.type}
    />
  )
})
```

```tsx QRCode.tsx
const { wcUri, isFetchingWcUri } = useAppKitWallets()

if (isFetchingWcUri) {
  return <div>Getting the connection QR...</div>
}

if (wcUri) {
  return <QR value={wcUri} />
}
```

</CodeGroup>
