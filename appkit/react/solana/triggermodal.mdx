To open the modal you can use our [**React components**](../core/components) or build your own button with Reown [**hooks**](../core/hooks.mdx#useAppKit).

<Tabs>
<Tab title="React Component">

```tsx
import { AppKitButton } from "@reown/appkit/react";

export default function ConnectButton() {
  return <AppKitButton />;
}
```

Learn more about the Reown React components [here](../core/components)

</Tab>
<Tab title="Hooks">

You can trigger the modal by calling the `open` function from `useAppKit` hook.

```tsx
import { useAppKit } from "@reown/appkit/react";

export default function ConnectButton() {
  // 4. Use modal hook
  const { open } = useAppKit();

  return (
    <>
      <button onClick={() => open()}>Open Connect Modal</button>
      <button onClick={() => open({ view: "Networks" })}>
        Open Network Modal
      </button>
    </>
  );
}
```

Learn more about the Reown hooks [here](../core/hooks)
