<Note>
  When using AppKit components in Nuxt, make sure to wrap them in `<client-only>` tags for SSR compatibility.
</Note>

```javascript
<template>
  <client-only>
    <appkit-button />
    <appkit-network-button />
    <appkit-connect-button />
    <appkit-account-button />
  </client-only>
</template>
```
